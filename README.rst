==============================================================================
Plone Frontpage Content-Type
==============================================================================

A configurable Frontpage Content-Type for Plone. 

Installation
------------

Install operun.frontpage by adding it to your buildout::

    [buildout]

    ...

    eggs =
        operun.frontpage


and then running ``bin/buildout``
